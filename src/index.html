Projects

Plant Disease Detection

The aim of this project was to develop a android app that utilizes deep learning techniques to predict plant diseases using computer vision. The app is designed specifically for farmers in rural areas who do not have access to network connectivity and are using cheap Android handsets. The deep learning model which serves as the backend of the app to provide it's offline predictive capabilities is trained useing transfer learning on a base model of mobilenet trained on imagenet. The app provides a user-friendly interface for farmers to take a picture of a plant and get an immediate offline diagnosis of the disease. In conclusion, this project demonstrates the potential of using deep learning and mobile apps to solve real-world problems and help farmers in rural areas.

Technologies used: Pytorch, Android SDK
